# IRONWORLD — Empire of the Iron God

A god-sim powered by real gym sessions. Train → earn Power → build your island, conquer nations, raise mortals into Champions, and write one growing Saga.

Built with [Vite](https://vitejs.dev/) + React. Save data lives in the browser via `localStorage`.

## Run locally

```bash
npm install
npm run dev
```

Then open the URL Vite prints (default http://localhost:5173).

## Build for production

```bash
npm run build      # outputs to dist/
npm run preview    # serve the production build locally
```

## Deploy

The app is a fully static site after `npm run build`. Deploy the `dist/` folder to any static host (Vercel, Netlify, GitHub Pages, Cloudflare Pages, etc.).
